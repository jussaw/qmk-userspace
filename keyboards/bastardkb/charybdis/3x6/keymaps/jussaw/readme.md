# jussaw's Charybdis 3x6 keymap

Colemak-DH base with home-row mods, shared with every other `jussaw` board via
the `users/jussaw/` userspace: Lower (symbols), Upper (numbers + F-keys + nav),
Adjust (media, mouse keys, `QK_BOOT`; hold Lower + Upper), Settings, and a
toggleable Game layer. Two board-local overrides, both forced by its 5-key thumb
cluster: the trackball DPI/sniping/drag-scroll keys replace the mouse/lock
cluster on the Lower left half, and Backspace stays on the Default R1 edge.
