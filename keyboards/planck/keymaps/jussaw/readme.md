# jussaw's Planck keymap

Colemak-DH base with home-row mods, shared with every other `jussaw` board via
the `users/jussaw/` userspace: Lower (symbols + mouse/lock cluster), Upper
(numbers + F-keys + nav), Adjust (media, mouse keys, `QK_BOOT`; hold Lower +
Upper), Settings, and a toggleable Game layer. The planck has no thumb cluster,
so its 12-wide bottom row carries the thumb functions. The upstream default's
Muse/MIDI/encoder scaffolding has been removed.
